# Linkedin
A fully functional LinkedIn-style social networking platform built with PHP, MySQL, and Bootstrap. Users can sign up, log in, create posts (with images), like, comment, and manage their profiles.

A simple social networking web app similar to LinkedIn.  
Built using **PHP**, **MySQL**, and **Bootstrap 5**.

## Features
- User Signup & Login
- Create, Edit, Delete Posts
- Upload Images with Posts
- Like & Comment System
- User Profile Management

## Technologies Used
- Frontend: HTML, CSS, Bootstrap
- Backend: PHP
- Database: MySQL

## Setup
1. Import `database.sql` in your MySQL.
2. Update database credentials in `includes/config.php`.
3. Run the project via `http://localhost/linkedin_clone/`.
